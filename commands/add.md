# /codebase-router:add

Fügt eine neue Area manuell zum Routing hinzu.

## Ablauf
1. Frage: "Name der Area? Welcher Ordner?"
2. Scanne den Ordner, identifiziere Files und Nodes
3. Erstelle area/_.md + Node-Files im .routing/ Format
4. Ergänze INDEX.md um den neuen Eintrag
5. Bestätigung: "Area '[name]' mit [X] Nodes hinzugefügt."
